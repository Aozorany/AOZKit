# AOZKit
一系列在我自己工程里面使用的工具集
## 系统要求
* Xcode 8+；
* iOS 7+；
* MRC 或 ARC；
* Python环境。
## 如何使用
把AOZKit目录下的h和m文件加入到你自己的工程中，并import AOZKit.h。
## 主要功能
* aoz_dispatch_once：带有一个object和一个block，可以保证在object的生命周期内，block只执行一次；
* GM_DEBUG_TIMER：用于记录并输出代码段执行时间的宏，只在DEBUG环境下有用；
* UIView及其子类的链式写法。
